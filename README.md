# Задание 3

## Задача 1

### Текст задачи

Опишите функции, осуществляющие основные математические операции над комплексными 
числами (сложение, вычитание, умножение, деление, возведение в степень). Использовать 
готовый класс Complex запрещено.

### Алгоритм решения

1. Вспомнить тонкости работы с комплексными числами (Опционально)
2. Создание своего типа данных (Содержит Re - действительная часть числа; Im - мнимая часть числа)
3. Функция сложения: На вход подается два компелксных числа - возвращается результат по формуле (a+c) + (b+d)i
4. Функция вычитания: На вход подается два компелксных числа - возвращается результат по формуле (a-c) + (b-d)i
5. Функция умножения: На вход подается два компелксных числа - возвращается результат по формуле (ac-bd) + (ad+bc)i
6. Функция деление: На вход подается два компелксных числа - возвращается результат по формуле ((ac+bd) / c**2 + d**2) + ((bc-ad) / c**2 + d**2)i. Для удобства (с**2 + d**2) просчитывается заранее
7. Функция возведение в степень: На вход подается одно комплексное число и степень N в которую нужно возвести - возвращается результат по формуле r^n * (cos(n*φ) + i*sin(n*φ)). r - берется по модулю, угол φ высчитывается при помощи встроенной функции atan2, r^n - r возводится в степень n
